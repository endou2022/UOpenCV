コンストラクタ

UOpenCV(filename=None)
コンストラクタ
filename ファイル名を指定すると、オブジェクトを作る時に読み込む。

プロパティー

bitmap
ビットマップデータ 書き込みたい場合は _bitmap を利用する
読み出し専用プロパティー
name
ファイル名 読み込み時に設定される
読み出し専用プロパティー
kind
画像種別 enum('no_data','color','gray','binary','fft',...)
読み出し専用プロパティー
width
画像の幅
読み出し専用プロパティー
height
画像の高さ
読み出し専用プロパティー
log
処理履歴 リスト形式で処理した履歴を記録している
読み出し専用プロパティー

入出力

imread(filename: str)
画像入力 読み込めるファイルは cv2.imdecode の許す形式 Windows bitmaps, JPEG, Portable Network Graphics, WebP 読めない場合は例外を発生 gifファイルは読み込めない cv2.imreadはパスに日本語が入ると読み込めない(ver.4.5)ので、numpy を利用してファイルの読み込みをしている
filename: 画像ファイル名
img = UOpenCV()
try:
    img.imread('lena.png')
except Exception as e:
    print(e)
imwrite(filename: str)
画像出力 ファイルに保存　形式は拡張子で自動的に判別される。PNG形式を推奨 書き込めない場合は例外を発生 cv2.imwriteはパスに日本語が入ると画像名が文字化けするので、cv2.imencode を利用してフォーマットを整え、ファイル出力している
filename: 画像名 ディレクトリを作ることはできない
