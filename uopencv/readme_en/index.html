Features

  • UOpenCV is a wrapper class for OpenCV. Some of the functions of opencv-python (+ image processing in books and homepages) have been made into classes to make them easier to use (for the author).
  • This is a class created to create image processing software. Since it is independent of image processing software, you can create various software using classes.
  • Most functions return an image object (UOpenCV), so you can create a method chain.
  • As a general rule, methods are called with name arguments.
  • Most function parameters have been adjusted and can be used with the defaults.
  • I have confirmed the operation, but the author does not take any responsibility even if any problem occurs in this program. (Author's disclaimer)
