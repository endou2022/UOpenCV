Input / output

imread (filename: str)
Image input The file that can be read is the format allowed by cv2.imdecode. Windows bitmaps, JPEG, Portable Network Graphics, WebP Raises an exception if unreadable gif files cannot be read cv2.imread cannot be read if Japanese is entered in the path (ver.4.5), so numpy is used to read the file.
filename: image filename
img = UOpenCV ()
try: try:
    img.imread ('lena.png')
except Exception as e:
    print (e)
imwrite (filename: str)
Image output Save to file The format is automatically determined by the extension. PNG format recommended Raises an exception if not writable In cv2.imwrite, the image name is garbled when Japanese is entered in the path, so format is adjusted using cv2.imencode and output as a file
filename: image name You can't create a directory
