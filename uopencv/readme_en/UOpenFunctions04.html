threshold (thresh: int = 128, type: int = cv2.THRESH_BINARY)
Global binarization If it is a color image, it is grayscaled and binarized. If it is a grayscale (1 channel) image, it will be binarized as it is
threshold when thresh cv2.THRESH_BINARY type Binarization method cv2.THRESH_BINARY threshold Set the value below 0 to 0 and the other values ​​to maxValue (255) to binarize. cv2.THRESH_OTSU Thresh that automatically determines the threshold by Otsu's method is ignored cv2.THRESH_TRIANGLE Triangle algorithm automatically determines threshold thresh is ignored
UOpenCV binarized image object UOpenCV.threshold Threshold when calculated automatically
adaptiveThreshold (adaptiveMethod: int = cv2.ADAPTIVE_THRESH_MEAN_C, thresholdType: int = cv2.THRESH_BINARY, blockSize: int = 51, C: float = 0)
Adaptive binarization If it is a color image, it is grayscaled and binarized. If it is a grayscale (1 channel) image, it will be binarized as it is. Calculate the threshold value in a certain neighborhood area and perform binarization processing in each area
adaptiveMethod Algorithm used for adaptive thresholding cv2.ADAPTIVE_THRESH_MEAN_C, cv2.ADAPTIVE_THRESH_GAUSSIAN_C cv2.ADAPTIVE_THRESH_GAUSSIAN_C feels more natural, but considering labeling, cv2.ADAPTIVE_THRESH_MEAN_C seems to be better. thresholdType Binarization type cv2.THRESH_BINARY, cv2.THRESH_BINARY_INV blockSize Neighborhood size for threshold calculation Odd greater than 1 C Calculated threshold minus C is used as the final threshold
