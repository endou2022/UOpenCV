Inverse FFT transform

idft (mask = None)
Inverse FFT transform
mask mask image If it is not a binary image, it is binarized internally. If not specified, only reverse conversion is performed
Mask image must be the same size image

Inter-image operation

addWeighted (alpha: float, src2, beta: float, gamma: float)
Image blend Images must be the same size and have the same number of channels Calculate dst = self * alpha + src2 * beta + gamma It is not necessary to add alpha and beta to 1
alpha weight UOpenCV src2 Arithmetic image object beta weight gamma offset
Return color image object
operation (src2, op: str)
Inter-image operation
src2 Image to be calculated op Operation The types of operation are as follows. ADD SUB MUL DIV ABS MAX MIN AND AND OR XOR
Return color image object
